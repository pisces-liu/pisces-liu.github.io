JS 5.数组

发表于2020-12-20|更新于2021-03-12|前端工程师 2020
|阅读量:

内容简介：1.数组的定义和基本使用，2.数组的常用方法，3.数组算法，4.引用类型

什么是数组

编程时，我们经常会遇见处理“一组数值”的情况，比如6个学生的考试成绩：

数组（Array）,顾名思义，用来储存一组相关的值，从而方便进行求和，计算平均值，逐项遍历等操作。

每种高级语言中都有数组，它是一种非常重要的数据结构。

数组的定义方法

  1. 直接使用方括号[ ]即可

  2. 使用构造函数来定义数组

  3. 创建一个已知长度的数组，但是这些项的值都是undefined.

访问数组项

数组的每一项都有下标，下标从 0 开始

可以使用方括号中书写下标的方式，访问数组中的某一项。

下标越界

JavaScript 规定，**访问数组中不存在的项会返回undefined**，不会报错

数组的长度

数组的 length 属性代表它的长度。数组最后一项的下标是数组的长度减1。

更改数组项

数组并不是只读的，我们可以修改它其中任何一项的值。

如果更改的数组项超过了length-1，则会创造该项

数组的遍历

数组的最大的优点就是方便遍历

数组类型的检测

  1. 数组用 typeof 检测结果是 object

  2. Array.isArray() 方法可以用来检测数组。

数组的头尾操作

方法 功能
push() 在尾部插入新项
pop() 在尾部删除
unshift() 在头部插入新项
shift() 在头部删除

push() 方法

  1. push() 方法用来在数组末尾推入新项，参数就是要推入的项。
  2. 如果要推入多项，可以用逗号隔开。
  3. 调用push() 方法后，数组会立即改变，不需要赋值

pop() 方法

  1. 与 push() 相反，pop() 方法用来删除数组中的最后一项。
  2. pop() 方法不仅会删除数组末项，而且会返回被删除的值。

unshift() 方法

  1. unshift() 方法用来在数组头部插入新项，参数就是要插入的项。
  2. 如果要推入多项，可以用逗号隔开。
  3. 调用 unshift() 方法后，数组会立即改变，不需要赋值

shift() 方法

  1. 与 unshift() 相反，shift() 方法用来删除数组中下标为0的项。
  2. shift() 方法不仅会删除数组首项，而且会返回被删除的项。

splice() 方法

splice() 方法用于替换数组中的指定项

splice() 方法用于在指定位置插入新项

splice() 方法用于删除指定项

slice() 方法

  1. slice() 方法用于得到子数组，类似于字符串的slice() 方法。
  2. slice(a, b) 截取的子数组从下标为a的项开始，到下标为b（但不包括下标为b的项）结束。
  3. slice(a, b) 方法不会更改原有数组。
  4. slice() 如果不提供第二个参数，则表示从指定项开始，提取所有后续所有项作为子数组。
  5. slice() 方法的参数允许为负数，表示数组的倒数第几项。

join() 和 split() 方法

  1. 数组的 join() 方法可以使数组转为字符串；字符串的split() 方法可以使字符串转为数组

  2. join() 的参数表示以什么字符作为连接符，如果留空则默认以逗号分隔，如同调用toString() 方法。

  3. split() 的参数表示以什么字符拆分字符串，一般不能留空。

字符串和数组更多相关性

字符串也可以使用方括号内写下标的形式，访问某个字符，等价于charAt() 方法。

concat() 方法

concat() 方法可以合并连接多个数组，concat() 方法不会改变原数组。

reverse() 方法

reverse() 方法用来将一个数组中的全部项顺序置反

将字符串倒序

indexOf() 和 includes() 方法

  1. indexOf() 方法的功能是搜索数组中的元素，并返回它所在的位置，如果元素不存在，则返回-1.
  2. includes() 方法的功能是判断一个数组是否包含一个指定的值，返回布尔值。

遍历相关算法

  1. 求数组中的每一项的总和，平均值。

  2. 求数组项的最大值和最小值。

数组去重

题目：去掉数组中的重复项。

思路：准备一个空结果数组，遍历原数组，如果遍历到的项不在结果数组中，则推入结果数组。

随机样本

题目：请随机从原数组中取3项.

思路：准备一个空结果数组，遍历原数组，随机选择一项，推入结果数组，并且将这项删除。

冒泡排序

  1. 冒泡排序是一个著名的排序算法，也是在面试时非常爱考的算法。
  2. 冒泡排序的核心思路是一趟一趟地进行多次项的两两比较，每次都会将最小的元素排好位置，如同水中的气泡上浮一样。
  3. n 个数字，共需比较 n-1 次，比较次数为 n(n-1)/2 次。

二维数组

二维数组：以数组作为数组元素的数组，即“数组的数组”。二维数组可以看作矩阵。

基本类型和引用类型

基本类型：number, boolean, string, undefined, null.

引用类型： array, object, function, regexp

相等判断时的区别

  1. 基本类型进行相等判断时，会比较值是否相等。

  2. 引用类型进行相等判断时，会进行比较地址是否相等。

深克隆和浅克隆

  1. 使用 arr1=arr2 的语法不会克隆数组。
  2. 浅克隆：只克隆数组的第一层，如果是多维数组，或者数组中的项是其他引用类型值，则不会克隆其他层。
  3. 深克隆：克隆数组的所有层，要使用递归。

浅克隆案例：

如果一个数组里面只有基本类型的话，使用浅克隆也可以克隆一份数组。可是假如数组里面有引用类型的话，浅克隆就起不到作用了。
