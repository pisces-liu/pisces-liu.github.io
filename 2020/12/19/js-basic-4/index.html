for 在英语中没有特别的含义，其名称来源于1950年代的早期汇编语言，现在几乎世界上所有编程语言中都用 for 表示循环语句。

for 语句的语法

for的圆括号中有三个表达式：

  1. 表达式var i = 1; 表示定义一个“循环变量” i，并赋值为 1；
  2. 表达式i <= 10; 表示继续执行循环的条件，只要这个条件为真，则会一致执行；
  3. 表达式i++用来更新循环变量，是循环变量的值月俩月趋向重点。

for 语句的执行机理

  1. 首先会执行 var i = 1;
  2. 然后判断 i <= 10; 是否满足，如果满足则进入循环体，执行语句console.log(i),如果不满足则退出循环。
  3. 循环体中的语句执行完毕后，执行语句i++
  4. 再次判断i <= 10;是否满足，如果满足则进入循环体，执行console.log(i),如果不满足则退出循环。

while 循环语句

while 语句也是一种循环结构，是一种“不定范围”循环，和for循环各有不用的用武之地。几乎所有的编程语言，都同时提供for循环和while 循环。

while 循环的语法

while 语句事先不指定循环开始，结束的范围，只要测试条件满足，就一直执行循环体。

while 循环注意事项

while 循环没有显示定义循环变量，必须自己在while 循环外先定义好循环变量，有时甚至可以没有循环变量。循环体内的语句，必须使循环测试条件趋向不成立，否则会死循环。

break

break 表示立即终止循环，它只能用在循环语句中，在for循环和while循环中都可以使用。

break 用在 while 语句中，通常和 while(true){} 搭配使用

continue

continue 用于跳过循环中的一个迭代，只继续执行循环中的下一个迭代。for 循环更经常使用continue

什么是算法

  1. 算法（Algorithm）是指解决方案的准确而完整的描述，是一系列解决问题的清晰指令，算法代表着用系统的方法描述解决问题的策略机制。也就是说，能够对一定规范的输入，在有限时间内获得所要求的输出。

  2. 算法就是把一个问题，拆解为计算机能够一步一步执行的步骤。

  3. 计算机的流程控制语句：循序执行，选择语句，循环。

优秀算法的要求

  1. 正确性
  2. 健壮性
  3. 可读性

穷举法是什么

  1. 计算机最突出的能力是计算，它没有总结归纳，逻辑推理的能力。所以人们使用计算机解决问题的时候，要“扬长避短”–充分发挥计算机的计算优势，而不要让他进行逻辑推理。穷举法就是这样的一种算法思想。
  2. 穷举法，顾名思义，是指根据题目的条件确定答案的大致范围，并在此范围内对所有可能的情况逐一验证直到全部情况验证完毕，
