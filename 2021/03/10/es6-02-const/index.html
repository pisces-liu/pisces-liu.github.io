在搞清楚这个问题之前，我们先来看一下 JS 储存变量的方式。

JS 变量的存储方式：在 JS 里面变量的存储方式有两种，一种是储存在栈内存（stack）里面，比如 number , boolean, string, undefined, null , Symbol；另外一种是储存在堆内存（heap）里面，比如 object 。栈内存里面也会存储引用类型值，但是其存储的是引用类型的地址，也就是指向堆内存的地址，而不是其真正的值。

回到刚刚的问题中，const 声明的常量不能改变，准确的来说是常量的内存地址不能被改变。当我们修改被 const 声明的引用类型的常量的时候，我们修改的只不过是地址里面存储的值，而不是修改常量的地址。
